URL specification update area

We have been working on a revision of RFC 1738 (Uniform Resource Locators) and RFC 1808 (Relative Uniform Resource Locators). This is my active work area.

The work is progressing as follows:

  1. I grabbed the current set of URL RFCs and removed the header/footer garbage that makes diff comparisons difficult.
  2. I created a base document that merges RFC 1738 and RFC 1808 without changing any of the effective content, except in that all of the scheme-specific definitions are removed.
  3. I created test pages for relative URL parsing, parts one, two, three, four, and five, and have tested a few common browsers.
  4. I wrote the first three drafts (changes noted in the last appendix):
    • draft-fielding-url-syntax-00.txt
    • diff from 00 to 01, draft-fielding-url-syntax-01.txt
    • diff from 01 to 02, draft-fielding-url-syntax-02.txt
